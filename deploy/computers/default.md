# eCat &mdash; эмулятор 8-разрядных компьютеров

Проект создан в рамках поддержки русскоязычной энциклопедии эмуляции [Emuverse.ru](https://emuverse.ru).

---

Выберите интересующий вас компьютер в панели слева — справа отобразится краткая справка, а также ссылки, по которым можно найти дополнительные материалы и загрузить программное обеспечение.


Для каждого компьютера предусмотрен минимальный набор программ, достаточный для его запуска и начального использования.

Подробная документация доступна на странице проекта на [GitHub](https://github.com/Ptr314/ecat3).
