# eCat &mdash; эмулятор 8-разрядных компьютеров

Проект создан в рамках русскоязычной энциклопедии эмуляции [Emuverse.ru](https://emuverse.ru).

---

Выберите интересующий вас компьютер в панели слева — справа отобразится краткая справка, а также ссылки на дополнительные материалы, где можно подробнее ознакомиться с его историей и возможностями, а также загрузить программное обеспечение.


Для каждого компьютера предусмотрен минимальный набор программ, достаточный для его запуска и первичного использования.
